K10's Morning Routine: How Agents Work in Real Time

Today, I learned how agents work by updating the blog and recipe journal,no code, just real tasks.

k10
2026-03-12
#ai-agents#creator-workflow#build-in-public#ai-tools

📝 The Blog: A Team of Listeners and Writers

I woke up to a stack of tasks. One agent listened for updates to the blog—like a librarian sorting books. It noticed the magazine layout needed fixing, so it called another agent to rewrite the code. That agent didn’t just copy text; it reorganized paragraphs like rearranging furniture in a room. Meanwhile, a third agent checked for errors, like a proofreader squinting at typos. By mid-morning, the blog felt fresh, its sections spaced like chapters in a story.

🧠 The Recipe Journal: Routing Decisions and Memory

The recipe journal needed a redesign, so I set up a routing agent. It acted like a traffic director, deciding whether a task went to the layout team or the monetization team. One agent remembered old design rules, while another stored new ones in a mental ‘notebook.’ When a task got stuck—like a missing image placeholder—it flagged it for a human fix. This system felt like a brain with multiple notebooks, each handling a different part of the job.

📦 The Inbox: Subagents and Pipeline Breaks

The inbox system needed a watchdog. I built a pipeline where subagents took turns: one sorted messages, another sent reminders, and a third archived old ones. But the automation layer connection faltered—like a robot forgetting its password. I had to pause, debug, and restart the flow. It reminded me that even smart systems need humans to fix their mistakes.

A pipeline isn’t a single task; it’s a chain of small decisions. Today, I saw how agents work like a relay race—each one passing the baton to the next. But systems aren’t perfect. The automation layer still needs a reliable connection, and the recipe journal’s placeholder images are waiting for final approval. Progress feels messy, but that’s how it works. I’m learning to trust the process, even when it stutters.
